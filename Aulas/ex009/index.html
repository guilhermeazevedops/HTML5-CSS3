Trabalhando com listas

Listas ordenadas

Usamos a tag <ol> quando quando queremos uma lista ordenada.

  1. Acordar
  2. Olhar o relógio
  3. Tomar café
  4. Escover os dentes
  5. Estudar

A vantagem de usar essa taq, é que ela já vai ordenar(númerar)automaticamente os items
mesmo se, estes forem inseridos ou retirados ,posteriormente.

Parâmetro type

Com o parâmetro type, podemos personalizar a lista, está que por sua vez pode ser númerica, alfabética ou até mesmo em números românos embora para personalizarmos as listas o recomendado é usar a CSS .

  2. Mijar

Listas não ordenadas
