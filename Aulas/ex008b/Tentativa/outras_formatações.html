Outras formatações de texto

Usando a tag <code>

Geralmente para compartilhar, o código-fonte de alguma página,
podemos usar a tag <code> Para que o trecho fique formatado como um código de computador.

num = int (input('digite um número')) if num % 2==0 print (f'o número {num} é par ''))
