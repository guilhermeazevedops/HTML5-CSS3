Formatando textos em HTML

Itálico e Negrito

Usando a tag <strong> (Negrito)

A tag <strong> é utilizada, quando queremos dar sentido de peso em certo trecho do nosso texto
trata-se de uma tag semânica,embora também temos outra tag a tag <b> não-semântica

Usando a tag <em> (Itálico)

Usamos a tag <em> (semântica) quando quereremos dar ênfase em certo trecho de nosso texto.
Também temos a tag <I> porém essa só possui forma.

Texto Marcado

Usando a tag <mark> (Marca-texto)

Usamos a tag <mark> quando queremos destacar (ou deixar anotado) parte do texto, o padrão é amarelo.
Contudo fazendo o uso de CSS podemos trocar a por qualquer cor.

Tamanhos de Texto

Usando a tag <big>

Embora a HTML seja uma linguagem focada em conteúdo, ainda sim, temos a tag <big>
não-semântica, que serve para deixar o texto grande E está esta deixando de ser usada e se tornou obsoleta

Usando a tag <small>

Da mesma forma da tag anterior a tag <small> serve para deixar um texto pequeno, embora parecida com a tag <big>
a tag small segue em uso e é considerada pela W3C semântica.

Texto deletado

Usamos a tag <del> quando parte do texto pode ser lido porém não precisa ser considerado

Texto inserido usando as tags <ins> e <u>

Texto inserido tag <ins>

Usamos a tag <ins> quando queremos, indicar que parte do texto foi
inserido depois. Um trecho importante que deve ser considerado
trata-se também de uma tag semântica.

Texto inserido tag <u>

Da mesma forma com a tag <ins> também temos a tag <u> não-semântica

Texto sobrescrito tag <sup>

Usamos a tag sup em formúlas matemáticas do tipo: x2+3 serve para subir

Texto subscrito

A tag <sub> é utilizada geralmente em fórmulas químicas do tipo h20 ou para descer parte do texto
